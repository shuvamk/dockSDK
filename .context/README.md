# .context — LLM Context Management

This directory maintains continuity across LLM coding sessions.

## For LLMs / AI Agents

- **Start of session**: Read `CURRENT_STATE.md` first, then `DECISIONS.md`
- **End of session**: Follow the instructions in `HANDOFF_PROMPT.md`
- **Before making architectural decisions**: Check `DECISIONS.md` for prior decisions

## For Humans

- `CURRENT_STATE.md` — What's the project doing right now
- `DECISIONS.md` — Why things are the way they are
- `ARCHITECTURE.md` — How everything fits together
- `CHANGELOG.md` — What changed and when
- `sessions/` — Detailed per-session logs (archival)
